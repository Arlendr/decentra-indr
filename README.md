# 🛡️ AI-Антикартель
### Система графового и семантического анализа госзакупок для выявления скрытых сговоров

> **Decentrathon 5.0 | Трек: AI inDrive | Кейс 3: AI for Government**

---

## 🎯 Проблема

В Казахстане ежегодно через государственные закупки проходят **триллионы тенге**. Аудиторов мало, тендеров — миллионы. Самая популярная схема — **картельный сговор**: три компании, принадлежащие одному владельцу, имитируют конкуренцию. Или чиновник «затачивает» техспецификацию под конкретного поставщика.

## 🤖 Что делает система

Три ML-модуля присваивают каждому тендеру **Risk Score (0–1)**:

| Модуль | Технология | Что ловит |
|--------|-----------|-----------|
| **Anomaly Detection** | Isolation Forest | Цены, завышенные в 10–200 раз |
| **Graph Analysis** | NetworkX | Компании с одним IP/телефоном, картельные кластеры |
| **NLP Analysis** | TF-IDF + Cosine Similarity | «Заточенные» спецификации, совпадение с КП картеля |

**Risk Score** = 0.35×Цена + 0.40×Граф + 0.15×NLP-схожесть + 0.10×NLP-паттерны

## 📊 Данные

Система работает на **синтетически сгенерированных данных** со встроенными коррупционными паттернами:
- 40 нормальных тендеров
- 8 картельных тендеров (3 компании, совпадающие IP/телефоны)
- 5 тендеров с аномально завышенными ценами (×50–200 от рыночной)
- 3 тендера с «заточенными» спецификациями

В продакшене — интеграция с открытым API **goszakup.gov.kz**.

## 🔍 Explainability (объяснимость)

Для каждого тендера система выдаёт **человекочитаемое объяснение**:
- `🔴 Совпадение IP-адресов победителя и участника (10.0.0.99)`
- `🔴 Цена превышает среднерыночную в 120 раз`
- `🔴 Текст спецификации на 87% совпадает с коммерческим предложением ТОО «АлфаГрупп»`

AI **не заменяет** аудитора — только помогает сфокусироваться на 1% самых подозрительных закупок.

## ⚠️ Ограничения MVP

- Данные синтетические (не из реального goszakup.gov.kz)
- Граф связей строится только по IP и телефону (в реале — ещё учредители, родственники)
- NLP обучен на 3 эталонных КП (в реале — тысячи документов)
